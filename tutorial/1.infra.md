## 테라폼 설치 ##

* https://developer.hashicorp.com/terraform/install


## VPC 설치 ##

```
$ cd
$ git clone https://github.com/gnosia93/eks-grv-mig.git
$ cd eks-grv-mig/tf
$ terraform init
$ terraform apply --auto-approve
```

## kube config update ##

```
$ aws eks update-kubeconfig --region ap-northeast-2 --name eks-grv-mig
```
