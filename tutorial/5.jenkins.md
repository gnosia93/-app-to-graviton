도커 설치여부를 확인한다. 
```
$ sudo systemctl status docker
$ sudo docker run hello-world
```

젠킨스를 설치한다.
```
$ sudo docker run -d --name jenkins -p 8080:8080 jenkins/jenkins:jdk11
```
